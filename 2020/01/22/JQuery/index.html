    JQuery

    • jq是一个js语言的框架,对原生的js代码进行封装,可以让我们写的更少但实现的更多.
    • 如何引入jq框架,jq就是一个通过js语言所写的框架,实际上就是一个普通的js文件

    js对象和jq对象互相转换

    1. js转jq
    2. var jq = $(js);
    3. jq转js jq对象就是一个数组 var js = jq[0];

    选择器

    1. 基础选择器
    2. 标签名选择器:$("div)
    3. id选择器:$("#id")
    4. class选择器:$(".class")
    5. 分组选择器:$("div,#id,.class")
    6. 任意元素选择器:$("*")
    7. 层级选择器
    8. $("div span")匹配div里面的所有span
    9. $("div>span")匹配div里面的所有span子元素
    10. $("div+span")匹配div的弟弟元素span
    11. $("div~span")匹配div的弟弟们元素span
    12. 层级相关的方法
      1. 获取id为abc元素的所有兄弟元素$("#abc").siblings("div")
      2. 获取id为abc元素的哥哥元素$("#abc").prev("div")
      3. 获取id为abc元素的哥哥们元素$("#abc").prevAll()
      4. 获取id为abc元素的弟弟元素$("#abc").next()
      5. 获取id为abc元素的弟弟们元素$("#abc").nextAll()
      6. 获取id为abc元素的父元素$("#abc").parent()
      7. 获取id为abc元素的子元素们$("#abc").children()
    13. 过滤选择器
      1. $("div.first")匹配所有div中的第一个
      2. $("div.last")匹配所有div中的最后一个
      3. $("div:even")匹配所有div中下标为偶数的div
      4. $("div:odd")匹配所有div中下标为奇数的div
      5. $("div:eq(n)")匹配下标为n的div
      6. $("div:it(n)")匹配下标小于n的div
      7. $("div:gt(n)")匹配下标大于n的div
      8. $("div:not(.abc)")匹配class值不是abc的div
    14. 内容选择器
      1. $("div:has(p)")匹配包含p子元素的div
      2. $("div:empty")匹配空的div
      3. $("div:parent")匹配非空的div
      4. $("div:contains('xxx')")匹配包含xxx文本的div
    15. 可见选择器
      1. $("div:visible")匹配所有可见的div
      2. $("div:hidden")匹配所有隐藏的div
    16. 隐藏显示相关的方法
      1. $("#abc").hide()隐藏
      2. $("#abc").show()显示
      3. $("#abc").toggle()隐藏显示切换
    17. 属性选择器
      1. $("div[属性名]")匹配包含xxx属性的div
      2. $("div[属性名='值']")匹配xxx=xxx的div
      3. $("div[属性名!='值']")匹配xxx不等于xxx的div
    18. 子元素选择器
      1. $("div:first-child")匹配是div并且是第一个并且是子元素
      2. $("div:last-child")匹配是div并且是最后一个并且是子元素
      3. $("div:nth-child(n)")匹配是div并且是第n个并且是子元素
    19. 表单选择器
      1. $(".input")匹配表单中所有控件
      2. $(".password")匹配所有密码框
      3. $(".radio")匹配所有单选
      4. $(".checkbox")匹配所有多选
      5. $(".checked")匹配所有选中的单选,多选和下拉选
      6. $("input:checked")匹配所有选中的单选和多选
      7. $(".selected")匹配选中的下拉选
