    HTML

    Hhyper Text Markup Language超文本标记语言,不仅仅是纯文本(还包括字体相关和多媒体(图片,音频,视频))

    <! DOCTYPE html> 文档声明,告诉浏览器使用html5的标准解析页面
<html> 根标签
    <head> 头标签,给浏览器的看内容写在头标签里面
        <meta charset="utf-8"> 告诉浏览器字符集是utf-8
        <title>Insert title here</title> 页面标题
    </head>
    <body> 体标签,给用户看的内容写在body里面

    </body>
</html>

    学习HTML主要学习的就是有哪些标签,标签有哪些属性以及标签和标签之间的嵌套关系

    常见的文本标签

    1. 内容标题 <h1>-<h6> 独占一行,字体加粗,自带上下间距
    2. 段落标签 <p> 独占一行,自带上下间距
    3. 换行标签 <br> 在html页面中会回车不能换行,只能识别一个空格
    4. 水平分割先 <hr>
    5. 加粗 <b>
    6. 斜体 <i>
    7. 小字 <small>

    列表标签

    1. 无序列表: ul:type(显示图片) li
      <ul type="square"> <!--unordered list 无序列表-->
 <li>刘备</li> <!-- list item列表项 -->
 <li>关羽</li>
</ul>
    2. 有序列表: ol:type=(序号类型) start=(起始值) reversed=(降序) li
      <ol type="1" start="10" reversed="reversed"> <!-- ordered list有序列表-->
 <li>吃饭</li>
 <li>睡觉</li>
</ol>
    3. 定义列表 dl dt dd
      <dl> <!--定义一个列表-->
 <dt>凉菜</dt> <!-- 定义一个标题 -->
 <dd>拍黄瓜</dd> <!--定义描述 -->
 <dt>热菜</dt> <!-- 定义一个标题 -->
 <dd>乱炖</dd> <!-- 定义描述 -->
 <dd>热的</dd>
</dl>
    4. 列表嵌套: 有序和无序可以任意嵌套
      <ul>
 <li>凉菜
     <ol>
         <li>东北大拉皮</li>
         <li>果仁菠菜</li>
     </ol>
 </li>

     <ol>
         <li>番茄炒蛋
             <ul>
                 <li>甜口味</li>
                 <li>咸口味</li>
             </ul>
         </li>
         <li>锅包肉</li>
     </ol>
 </li>
</ul>

    图片标签 img

    • src: 路径 相对路径: 访问站内资源 图片和页面同一目录:直接图片名 图片在上级目录: ../图片名 图片在下级目录: 文件夹名/图片名 绝对路径: 访问站外资源,图片盗链,节省本站资源,有找不到图片的风险
    • alt: 图片不能显示时显示的文本
    • title: 图片标题,选题时显示
    • width/height: 两种赋值方式 像素和百分比(上级元素的百分比),如果只设置宽度,高度会等比例缩放
    • 支持的格式: jpe/jpeg png gif

    超链接 a

    • 如果不写href属性就是文本
    • href属性表示路径,可以是相对路径也可以是绝对路径,可指向页面资源也可以指向文件资源,指向资源时如果能浏览则浏览不能浏览则下载.
    • a标签包裹文本则时文本超链接,包裹图片则是图片超链接

    表格标签 table

    • table: border边框粗细 cellspacing单元格间距
    • tr 表示行
    • td 表示列
    • colspan 跨列
    • rowspan 跨行
    • caption 表格标题

    CSS的三种引入方式

    1. 内联样式: 在标签的style属性中添加css样式代码,弊端:不能复用
    2. 内部样式: 在head标签里面添加style标签,标签体内写样式代码,弊端:只能当前页面复用,不能多页面复用
    3. 外部样式: 在单独的css文件中写样式代码,在html页面通过link标签引入样式文件,可以多页面复用
    4. 优先级: 内联优先级最高,内部和外部优先级一样,后执行的覆盖先执行的

    选择器

    1. 标签名选择器: 格式: 标签名{样式代码}
    2. id选择器: 格式: #id{样式代码}
    3. 类选择器: 格式: .class{样式代码}
    4. 属性选择器: 格式: 标签名[属性名='值']{样式代码}
    5. 分组选择器: 格式: #id,.class,div{样式代码} 将多个选择器合并到一起称为分组选择器
    6. 子孙后代选择器: 格式: div div p{样式代码} 匹配div里面的div里面的所有p标签(包包括子元素p和所有后代p)
    7. 子元素选择器: 格式: div>div>p{样式代码} 匹配div里面的div里面的所有子元素p
    8. 伪类选择器: 选择的是标签的状态: 未访问状态:link 访问过状态:visited 悬停状态:hover 点击状态:active
    9. 任意元素选择器: 格式: *{代码样式}

    常用样式

    1. 背景图片
      /* 设置背景图片 */
backgroud-image: url("../imgs/a.jpg");
/* 图片尺寸 */
backgroud-size: 100px 100px;
/* 禁止重复 */
backgroud-repeat: no-repeat;
/* 设置位置 横向和纵向百分比*/
backgroud-position: 0% 100%;
    2. 显示方式 display

      block: 块级元素,独占一行,可以修改宽高, 包括: h1-h6,p,div等
      inline: 行内元素,共占一行,不能修改宽高, 包括: b,a,i,small,span
      inline-block: 行内块元素,共占一行也可以修改宽高, 包括: img

    3. 盒子模型
      1. 盒子模型 = 宽高+外边距+边距+内边距
      2. 盒子模型之宽高:

        赋值方式:width/height 2种: 1.像素 2.上级元素百分比
        行内元素不能修改宽高

    4. 盒子模型之外边距
      1. 什么是外边距: 元素距上级元素或相邻兄弟元素的距离
      2. 赋值方式: margin

        margin-left/right/top/botton:10px;
        margin:10px; 4个方向都是10px
        margin:10px 20px; 上下10px 左右20pxmargin:0 auto; 水平居中margin:10px 20px 30px 40px; 上 右 下 左 顺时针`

    5. 粘连问题: 当元素上边缘和上级元素的上边缘重叠时,给元素添加上外边距会出现粘连问题,给上级元素添加overflow:hidden解决
    6. 盒子模型之边框

      赋值方式: border: 边框粗细(像素) 边框样式 边框颜色;
      border-left/right/top/bottom: 边框粗细(像素) 边框样式 边框颜色;

    7. 盒子模型之内边距

      什么是内边距: 元素边缘距内容的距离
      赋值方式: padding-left/right/top/bottom:10px; padding:10px; padding:10px 0; padding:10px 20px 30px 40px;

    8. 盒子模型总结:
      1. 宽高: 控制元素的显示大小
      2. 外边距: 控制元素的显示位置
      3. 边框: 控制边框样式的
      4. 内边距: 控制内容位置

    文本和字体相关样式

    1. 水平对齐方式: text-align:left/right/center;
    2. 文本修饰: text-decoration:overline/underline/line-through/none(去掉文本修饰);
    3. 文本颜色: color:red;
    4. 文本阴影: text-shadow:颜色 x偏移值 y偏移值 浓度(值越大越模糊);
    5. 行高: line-height:30px;
    6. 字体大小: fint-size:20px;
    7. 加粗: font-weight:bold/normal;
    8. 斜体: font-style:italic;
    9. 字体设置: font-family: cursive,xxx,xxx,xxx;
    10. 字体大小+字体设置: font:20px xxx,xxx,xxx;

    CSS的三大特性

    1. 继承性: 元素可以继承上级元素文本的字体相关的样式,部分标签自带效果不受继承影响,比如:超链接的字体颜色,h1-h6字体大小
    2. 层叠性: 不同的选择器有可能选择到同一个标签,如果作用的样式不同,则全部层叠到一起全部生效,如果作用的样式相同则由优先级据顶顶
    3. 优先级: 各个选择器之间的优先级,作用范围越小,优先级越高.id>class>标签名>继承(属性间接选中)

    position定位方式: 4+1种

    1. position静态定位(文档流定位)

      元素默认的定位方式特点: 块级元素从上到下排列,行内元素从左向右排序
      格式:position:static;
      通过外边距控制元素所在位置

    2. position相对定位

      特点: 元素不脱离文档流
      格式:position:relative;
      通过left/right/top/bottom相对于元素初始位置做偏移

    3. position绝对定位

      position:absolute；position:relative绝对定位使用通常是父级定义position:relative定位，子级定义position:absolute绝对定位属性，并且子级使用left或right和top或bottom进行绝对定位。

    4. position固定定位

      固定定位：相对于屏幕视口（viewport）的位置！position: fixed;

    5. 浮动定位

      格式: float
      浮动的框可以向左或向右移动，直到它的外边缘碰到包含框或另一个浮动框的边框为止。由于浮动框不在文档的普通流中，所以文档的普通流中的块框表现得就像浮动框不存在一样。

    溢出设置

    hidden 超出隐藏
    visible 超出显示
    scrool 超出滚动显示
