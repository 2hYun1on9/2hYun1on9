    JDBC

    • java DataBase Connectivity : java数据库连接,JDBC是Sun工资提供用于java和数据连接的编程接口

      如何使用JDBC

      1. 创建Maven工程
      1. 在pom.xml文件中添加以下代码
      1. 创建Demo01 以下代码
           // 1.注册驱动
   Class.forName("com.mysql.jdbc.Driver");
   // 2.获取连接对象
   // 导包时选择java.sql
   Connection conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/newdb3","root","qaz123");
   System.out.println(conn);
   // 3.创建SQL执行对象
   Statement s = conn.createStatement();
   // 4.执行SQL
   String sql = "create table jdbc1(id int,name varchar(10))";
   s.executeUpdate(sql);
   // 5.释放资源
   conn.close();

    SQL执行对象 Statement

    • execute(sql) 此方式可以执行任何的SQL语句,但是建议执行DDL包括create,drop,alter等
    • executeUpdate(sql) 此方法用于执行增删改的SQL语句
    • executeQuery(sql) 此方法用于执行查询的SQL

    数据库连接池DBCP

    • DBCP: DataBase Connection Pool

    • 使用数据库连接池可以大大降低Web服务器和数据库服务器之间的连接次数从而提高执行效率

    • 使用方法: 1. 在pom.xml文件中添加jar包坐标

          <dependencies>
    <dependency>
        <groupId>mysql</groupId>
        <artifactId>mysql-connector-java</artifactId>
        <version>5.1.6</version>
    </dependency>
    <dependency>
        <groupId>commons-dbcp</groupId>
        <artifactId>commons-dbcp</artifactId>
        <version>1.4</version>
    </dependency>
</dependencies>
      1. 在 src/main/resources下面添加 jdbc.properties文件
        url=jdbc:mysql://localhost:3306/newdb3
username=root
password=qaz123

    • 3: 创建 DBUitls.java 内容如下:

        private static BasicDataSource ds;
  // 写到静态方法里面是为了让读取配置文件的代码
  // 和创建连接池的代码只执行一次
  static {
      Properties p = new Properties();
      InputStream ips = DBUtils.class.getClassLoader().getResourceAsStream("jdbc.properties");
      try {
          p.load(ips);
      } catch (Exception e) {
          e.printStackTrace();
      }
      String url = p.getProperty("url");
      String username = p.getProperty("username");
      String password = p.getProperty("password");

      // 换成连接池
      ds = new BasicDataSource();
      // 设置数据库连接信息
      ds.setUrl(url);
      ds.setUsername(username);
      ds.setPassword(password);
      ds.setInitialSize(3);
      ds.setMaxActive(5);
  }
  public static Connection getConn() throws SQLException {
      return ds.getConnection();
  }

    • 4.数据库池连接

        //创建数据库连接池对象

    登录功能实现步骤:

      1. 通过Scanner获取用户输入的username和password
      1. 通过getConn()代表获取连接对象
      1. 创建查询的SQL语句,里面的变量用?表示
      1. 通过已经创建好的SQL语句创建预编译的SQL执行对象
      1. 将用户输入的用户名和密码替换掉SQL中的?
      1. 执行SQL语句得到resultset对象
      1. while循环遍历resultset对象
      1. 获取查询到的数量 如果大于0输出登录成功 否则输出登录失败.
        Scanner scan = new Scanner(System.in);
   System.out.println("输入用户名:");
   String username = scan.nextLine();
   System.out.println("输入密码:");
   String password = scan.nextLine();
   try (Connection conn = DBUtils.getConn()){
       String sql = "select count(*) from user where username=? and password=?";
       PreparedStatement ps = conn.prepareStatement(sql);
       //替换 ?
       ps.setString(1, username);
       ps.setString(2, password);
       ResultSet rs = ps.executeQuery();
       while(rs.next()) {
           int count = rs.getInt(1);
           if(count>0) {
               System.out.println("成功");
           }else {
               System.out.println("失败");
           }
       }
   } catch (Exception e) {
       e.printStackTrace();
   }
}
}

    批量操作

    • 将多条SQL语句的多次数据传输合并成一次数据传输,从而提高执行效率
    • 如果SQL语句中有变量就用预编译SQL执行对象
        try (Connection conn = DBUtils.getConn()){
      String sql = "insert into user values(null,?,?)";
      PreparedStatement ps = conn.prepareStatement(sql);
      ps.setString(1,username);
      ps.setString(2,password);
      //添加到批量操作
      ps.addBatch(sql);
      //执行
      ps.executeBatch();
  } catch (Exception e) {
      e.printStackTrace();
  }
    • 如果没有变量使用Statement执行
        try (Connection conn = DBUtils.getConn()){
      String sql = "insert into user values(null,'aaa','aaa')";
      String sql1 = "insert into user values(null,'bbb','bbb')";
      String sql2 = "insert into user values(null,'ccc','ccc')";
      Statement s = conn.createStatement();
      //添加到批量操作
      s.addBatch(sql2);
      s.addBatch(sql1);
      s.addBatch(sql);
      //执行
      s.executeBatch();
  } catch (Exception e) {
      e.printStackTrace();
  }

    分页查询

        Scanner scan = new Scanner(System.in);
    System.out.println("请输入查询页数");
        int page = scan.nextInt();
        System.out.println("请输入查询的条数");
        int count = scan.nextInt();
        try (Connection conn = DBUtils.getConn()){
            //创建SQL语句
            String sql = "select username from user limit ?,?";
            //创建预编译SQL执行对象
            PreparedStatement ps = conn.prepareStatement(sql);
            ps.setInt(1, (page-1)*count);
            ps.setInt(2, count);
            ResultSet rs = ps.executeQuery();
            //遍历输出
            while(rs.next()) {
                String username = rs.getString("username");
                System.out.println(username);
            }
        } catch (Exception e) {
            e.printStackTrace();
        }

    获取自增主键值

        try(Connection conn = DBUtils.getConn()){
        String sql = "insert into user values(null,'Tom','123456')";
        Statement s = conn.createStatement();
        s.executeUpdate(sql,Statement.NO_GENERATED_KEYS);
        //获取自增主键
        ResultSet rs = s.getGeneratedKeys();
        while(rs.next()) {
            //得到返回的唯一ID值
            int id = rs.getInt(1);
            System.out.println(id);
        }
    }

    元数据

    • 数据库元数据
    • 表元数据
        try (Connection conn = DBUtils.getConn()){
      //获取数据库的元数据对象
      DatabaseMetaData dbmd = conn.getMetaData();
      System.out.println("数据库名称:"+dbmd.getDatabaseProductName());
      System.out.println("数据库驱动版本:"+dbmd.getDriverVersion());
      System.out.println("数据库用户名:"+dbmd.getUserName());
      //获取表元数据 需要先写查询
      String sql = "select * from emp";
      Statement s = conn.createStatement();
      ResultSet rs = s.executeQuery(sql);
      //得到表元数据对象
      ResultSetMetaData rsmd = rs.getMetaData();
      //得到表字段数量
      int count = rsmd.getColumnCount();
      //遍历每个字段名和类型
      for (int i = 0; i < count; i++) {
          String name = rsmd.getCatalogName(i+1);
          String type = rsmd.getColumnTypeName(i+1);
          System.out.println(name+":"+type);
      }
  } catch (Exception e) {
      e.printStackTrace();
  }
