    String

    • 字符串广泛应用 在 Java 编程中，在 Java 中字符串属于对象，Java 提供了 String 类来创建和操作字符串
    • String 用 ==比较则返回false,用 equals() 比较内容相同则返回true
    • 凡是String new了的 一定会在内存中开辟出来一个新的,尽管内容一样,返回true,如果比较值地址则是false
    • String 创建了多个字符串,如果内容相同 用== 比较则true;new了就是false
    返回值 方法和说明
    char charAt(int index); 获取当前字符串中指定下标的字符
    int String indexOf(int ch String str); 查找给定字符串在当前字符串的位置,若当前字符串不包含给定内容则返回值为-1
    int indexOf(int ch String str, int fromIndex); 返回指定字符第一次出现的字符串内的索引，以指定的索引开始搜索.
    int lastIndexOf(String str); 返回指定子字符串最后一次出现的字符串中的索引
    int lastIndexOf(int ch,int fromIndex); 在指定下标位置开始搜索最后一次出现的位置,从指定的下标开始向后搜索.
    int lenth(); 返回该字符串的数量
    boolean startsWith(String prefix); 判断是否以该字符串开始
    boolean endsWith(String prefix); 判断是否以该字符串结尾
    char[] toCharArray(); 将字符串存入char类型数组
    String subString(int start,int end); 截取当前字符串指定范围内的子字符串
    String subString(int beginIndex);从指定位置截取到末尾
    String trim(); 去除当前字符串两边的空白字符
    boolean matches(String regex); 告诉这个字符串是否匹配给定的正则表达式
    String[] split(String regex); 告诉这个字符串是否匹配给定的正则表达式
    String replaceAll(String regex, String replacement); 用给定的替换替换与给定的正则表达式匹配的此字符串的每个子字符串
    String concat(String str); 添加字符串到末尾
