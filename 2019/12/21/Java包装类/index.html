    包装类

    • 包装类是为了解决基本类型不能直接参与面向对象,开发问题而产生的,使得基本类型可以以对象的形式存在
    • 自动装箱特性:
        int a = 1;
  // 这里编译器在编译时会将代码改成
  // Integer i = Integer.valueOf(a);
  Integer i =a;
    • 自动拆箱特性:
        // 代码触发了自动拆箱特性.
  // a = i.intValue();
  a = i;
      返回值 方法和说明
      int intValue();将包装类转换成基本类型
      int parseInt(String s); 将字符串解析成int类型(前提必须是字符串内容必须是相同类型 否则会抛出异常)
      double parseDouble(String s);
      Object valueOf(Object o); 将基本类型转换成包装类
